# FloraVision - Plant E-Commerce Website

## Deployed Link - https://floral-website-2.onrender.com/

A modern, interactive plant e-commerce website built with Next.js 13+ and Tailwind CSS, featuring 3D plant galleries, top-selling plants, customer reviews, and more.

## Features

- plant showcase with 3D animations and motion effects
- Beautiful card-based design with emerging plant images and glitter border effects
- Top selling plants collection 
- Customer reviews carousel
- Best O2 Plants showcase
- 3D plant gallery with mouse-tracking and rotation effects
- Responsive mobile-first design
- Dark green premium theme with glassmorphism

## Tech Stack

- Framework: Next.js 
- Styling: Tailwind CSS
- Components: React functional components

## Instructions to run the project

Install dependencies
- pnpm install

Run development server
- npm run dev
- Open http://localhost:3000 to see the site.

Build for production locally 
- pnpm build
- pnpm start
